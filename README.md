### 这是一个KitX插件。可用于在桌面显示课表（可用于希沃白板）

## 如何打包？
 - 在项目文件目录运行dotnet bulid
 - 将LoaderStruct.json和PluginStruct.json复制到生成目录（...\bin\Debug\net6.0-windows）
 - 按照KitX的教程运行kxper打包工具
 - 将.kxp文件拖入KitX Dashboard的“仓库”页
 - 点击运行
